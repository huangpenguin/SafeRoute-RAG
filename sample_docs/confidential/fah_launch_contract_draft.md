# 【合成デモ・非公開】FAH 法務部リーガルチェック報告：海外発射受託契約における契約漏洞リスク



## 1. 背景とドキュメントの目的


本ドキュメントは、極東航空宇宙ホールディングス（FAH）法務部およびリスク管理委員会の指定メンバーのみが閲覧可能な「社内機密」です。現在、欧州の巨大通信企業「AstroNet-EU」と締結交渉の最終段階にある、第5世代通信衛星コンステレーションの打ち上げ受託契約（社内プロジェクト・コード：シリウス）において、当社の経営を揺るがしかねない重大な「契約漏洞」が発見されました。至急の注意喚起と対応方針の共有を目的とします。



## 2. 契約漏洞の具体的な内容と法的リスク


先方から昨日提示された最新の契約ドラフト（Rev.4.2）の第18条「打ち上げ失敗時の免責と損害賠償責任」において、当社の免責条項の適用範囲が極めて限定的に、かつ意図的に書き換えられていることが判明しました。


特に、「不可抗力（Force Majeure）」の定義条項から「サプライチェーンに起因する予期せぬ部品供給の遅絶」および「天候以外の外的要因による発射ウィンドウの喪失」が完全に除外されています。


この「契約漏洞」を放置したまま担当役員が署名（サイン）した場合、もし当社のティア2以下の下請けサプライヤーで製造遅延が発生し、指定された打ち上げウィンドウを逃した際、FAHが最大で4億5,000万ユーロ（約720億円）の違約金および遅延損害賠償を全額負担する法的リスクが生じます。



## 3. 今後の対応方針と情報統制


至急、欧州の国際法務アドバイザー（法律事務所）を交えて、本条項の再修正（レッドラインのフィードバック）を実施します。この交渉が解決するまでは、関連する一切の「未公開」情報を社外に漏洩させないでください。万が一、この不利な契約交渉の事実がメディアや競合他社に流出した場合、当社の株価および今後の海外展開に深刻な影響を及ぼします。
